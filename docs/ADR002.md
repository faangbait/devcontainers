# ADR002

Ignore everything in ADR001. Start at the beginning. Don't even start at the beginning -- start at a scratchpad (`AS builder`) that comes several layers before the beginning. You built the scratchpad for one thing. Get a release of ripgrep, like so:

```dockerfile
FROM registry.access.redhat.com/ubi10:latest AS builder
ADD --unpack=true https://github.com/BurntSushi/ripgrep/releases/download/15.2.0/ripgrep-15.2.0-x86_64-unknown-linux-musl.tar.gz /tmp

FROM something
COPY --from=builder /tmp/ripgrep-*/rg /usr/local/bin/
```

Whenever possible: `FROM` early, `COPY` late. It's your sentinel. Everything before the `COPY` line: ground truth. Never altered. Or as my grandfather used to say, *SLOPPY BEFORE COPY, CACHE GETS DROPPIED*
